# Image Slider using CSS

This project is a simple image slider built using HTML and CSS. It showcases various images in a stylish, interactive manner, allowing users to navigate through them seamlessly.

## Live Demo

You can view the live demo of the image slider (https://im-prathamesh-dev.github.io/image-slider-using-css/).

## Features

- Responsive design that adapts to different screen sizes.
- Smooth transitions between images.
- Simple navigation controls for easy browsing.
- Pure CSS implementation for a lightweight experience.

## Technologies Used

- HTML
- CSS

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   bash
   git clone https://github.com/yourusername/image-slider-using-css.git
   
2. Navigate to the project directory:
   bash
   cd image-slider-using-css
   
3. Open index.html in your web browser.

## Usage

1. Open the project in a web browser.
2. Use the navigation arrows to slide through the images.
3. Enjoy the smooth transitions and responsive design!
